# Minecraft Tic Tac Toe datapack
## Installation
1. Go into your world folder > datapacks.

2. Git clone there

## Start
Go back in your world and if this pops up:

![alt text](https://i.imgur.com/yj7jyiQ.png)

press start! If it doesn't, do ``/reload``.
